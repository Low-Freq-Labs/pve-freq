# DC01 Cluster Automation Scripts

This directory will contain automation scripts for the DC01 3-node Proxmox VE cluster.

## Scope

Scripts in this directory target the in-scope infrastructure:
- Proxmox nodes: pve01 (10.25.0.26), pve03 (10.25.0.28)
- TrueNAS storage: 10.25.0.25
- Cisco WS-C4948E-F switch: 10.25.0.5 (via pve01 jump host)
- VMs 101-105

pve02, VM 100, and VMs 800-899 are out of scope.
